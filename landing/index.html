Everything you need

A real app for your .md files.

Open, read, edit, and share Markdown — beautifully, instantly, on macOS, Windows, and Linux.

Instant, beautiful rendering

GitHub-flavored Markdown, syntax-highlighted code, Mermaid diagrams, math, tables, footnotes — rendered the moment you double-click.

Edit without the syntax

A Typora-style WYSIWYG editor for everyone, plus a lossless source mode for power users. Saves are instant and safe.

Export anywhere

One click to a polished PDF, a Word doc, or a self-contained HTML page. Fully offline. No Pandoc, no accounts.

Smart agent output

Callouts, interactive checkboxes that save back to the file, and automatic plan / diff / multi-file detection — agent docs come alive.

Three gorgeous themes

Paper, Sepia, and Midnight — switch live, and everything from code to diagrams follows along. Beautiful in light and dark.

Native & instant

Built on Tauri — a tiny native binary that launches in a blink. No Electron bloat, no spinning beachballs.

Free · private · on-device

AI that never leaves your device.

Summarize, explain, rewrite, or translate any document — locally and free. On macOS 26+ (Apple Silicon): Apple's on-device models, zero setup. On Windows & Linux: Ollama, or bring your own key. Nothing uploaded by default.

  • ✓
    Genuinely free. Apple on-device models on Mac (zero setup) or Ollama on Windows & Linux — zero cost either way.
  • ✓
    Genuinely private. Your documents never touch a server on any platform. The green badge proves it.
  • ✓
    Tiered fallback. Apple Foundation Models → Ollama → your own cloud key. You always control where inference runs.

Agent-native

Your agents can drive it.

Ashlr MD speaks the language of coding agents. Open a file from the terminal, a URL, or let Claude Code read and edit the live document over MCP.

  • ›
    mdopen file.md — open from any terminal or script.
  • ↗
    mdopener:// — deep-link straight to a document.
  • ⚙
    MCP server — agents open, read, edit & export the current doc.
# open a file in Ashlr MD from anywhere
$ mdopen PLAN.md

# let Claude Code drive the app
$ claude mcp add mdopener mdopener-mcp
